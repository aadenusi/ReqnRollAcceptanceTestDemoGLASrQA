# Automated Test Scenarios for Greater London Authority

This document outlines the core automated test scenarios implemented for the GLA 

## Test Scenarios

#1. Search Result Validation
- **Valid search**: Users can carry out searches and their search result is relevant to keyword
---
The tests can be run on mobile views as well as desktop browsers to ensure responsiveness and usability across devices.

These scenarios are automated using Reqnroll, Selenium WebDriver and NUnit, following the Page Object Model for maintainability and clarity.
